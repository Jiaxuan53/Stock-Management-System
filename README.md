# 📦 Stock Management System (JavaFX GUI)

This project is a graphical user interface (GUI) application built with **JavaFX** for managing an inventory of electronic appliances. It provides an interactive system for users to initialize inventory, manage stock levels, and organize product data efficiently.

---

## ✨ Key Features

- **User Login**
  - Welcome screen requiring the user’s full name (first name and surname) to access the system.

- **Inventory Setup**
  - Users can define the number of products to be added during initial setup.

- **Dynamic Product Sorting**
  - View inventory sorted by:
    - Name (A–Z / Z–A)
    - Price (Low → High / High → Low)

- **Session Summary**
  - On exit, a termination screen is displayed showing:
    - User’s registered name
    - Generated user ID
    - A thank-you message

---

## 🏷️ Supported Product Categories

The system supports multiple appliance types, each with unique attributes:

- **Refrigerators**
  - Door design, color, capacity

- **Televisions (TVs)**
  - Screen type, resolution, display size

- **Washing Machines**
  - Type, load capacity, spin speed

- **Air Conditioners**
  - Type, horsepower, energy rating

---

## ⚙️ System Menu Operations

After setup, users can perform the following operations:

- **View Products**
  - Display all inventory items categorized by product type

- **Add Stock**
  - Increase stock quantity for selected active products

- **Deduct Stock**
  - Decrease stock quantity for selected active products

- **Discontinue Product**
  - Mark a product as discontinued (hidden from stock modification options)

- **Exit System**
  - Close the application and end the session

## 📌 Purpose

This project is designed to demonstrate object-oriented programming principles, GUI development using JavaFX, and inventory management system logic in a structured application.
